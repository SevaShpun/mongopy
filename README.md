#### MongoPy - примеры

```python
# Уникальный ID пользователя
user_id = 999

# База данных
dbase = DataBase(user_id)
# Регистрирует нового пользователя
dbase.Create()

# Получаем данные пользователя
users = dbase.user
# Изменяет значение ключа и добавляет если ключ не существует
users["age"] = 21
# Обновляет все введенные изменения
dbase.Update(users)
# Удаляет запись по ключу для указанного user_id
dbase.Delete("ark")
```
